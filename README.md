# Dev Quiz Android - Jetpack Compose

Este é um aplicativo de Quiz interativo desenvolvido em **Kotlin** com **Jetpack Compose**. O projeto foi criado como parte do exercício prático de desenvolvimento mobile, focado em gestão de estado, filtragem de dados e interfaces declarativas.

## 🚀 Funcionalidades

- **Seleção de Dificuldade:** O usuário escolhe entre os níveis Fácil, Médio ou Difícil antes de começar.
- **Filtragem Dinâmica:** O app carrega apenas as perguntas correspondentes ao nível selecionado.
- **Sistema de Pontuação:** Feedback em tempo real sobre o progresso e acertos.
- **Interface Moderna:** Desenvolvido inteiramente com Material Design 3 e componentes funcionais do Compose.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** [Kotlin](https://kotlinlang.org/)
- **Toolkit UI:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Arquitetura:** Componentes de função (Composables) e Gerenciamento de Estado (`remember`, `mutableStateOf`).
- **IDE:** Android Studio Panda / Ladybug.

## 📂 Estrutura do Projeto

O projeto segue os requisitos técnicos solicitados:
- `Pergunta.kt`: Contém a `data class Pergunta` e o `RepositorioDePerguntas` com a lógica de dados.
- `MainActivity.kt`: Contém a lógica de interface, controle de navegação entre telas (Início, Quiz e Final) e gestão de pontos.

## 📝 Requisitos Atendidos

- [x] Mínimo de 9 perguntas no repositório.
- [x] Pelo menos 3 perguntas por nível de dificuldade.
- [x] Validação de respostas e exibição de pontuação.
- [x] Interface construída 100% em código (sem XML).
